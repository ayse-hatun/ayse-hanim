## 🧠 System Thinking Approach

I do not treat LLMs as standalone intelligence.

I design systems where:

→ Input is structured  
→ Reasoning is constrained  
→ Output is validated  
→ Decisions are explainable  

### Core principle:
> LLMs are components inside systems — not the system itself.
